## 典型代码讲解

### 1. 直接插入排序

```c
void insert_sort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int temp = arr[i];  // 待插入元素
        int j = i - 1;
        // 从后往前找插入位置
        while (j >= 0 && arr[j] > temp) {
            arr[j + 1] = arr[j];  // 元素后移
            j--;
        }
        arr[j + 1] = temp;  // 插入到正确位置
    }
}
```

**关键逻辑**：
- 从第 2 个元素开始，依次插入前面已排序的部分
- 从后往前找插入位置
- 元素后移腾出空间

**示例**：
```
原数组：[5, 3, 1, 4, 2]

i=1: temp=3, 比较 5>3, 后移 → [3, 5, 1, 4, 2]
i=2: temp=1, 比较 5>1, 3>1, 后移 → [1, 3, 5, 4, 2]
i=3: temp=4, 比较 5>4, 后移, 3<4 停止 → [1, 3, 4, 5, 2]
i=4: temp=2, 比较 5>2, 4>2, 3>2, 后移 → [1, 2, 3, 4, 5]
```

### 2. 简单选择排序

```c
void select_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int min = i;  // 最小值下标
        // 找最小值
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min]) {
                min = j;
            }
        }
        // 交换
        if (min != i) {
            int temp = arr[i];
            arr[i] = arr[min];
            arr[min] = temp;
        }
    }
}
```

**关键逻辑**：
- 每轮找到最小值的位置
- 将最小值交换到前面
- 交换次数少

**示例**：
```
原数组：[5, 3, 1, 4, 2]

i=0: 最小值 1 (下标 2), 交换 → [1, 3, 5, 4, 2]
i=1: 最小值 2 (下标 4), 交换 → [1, 2, 5, 4, 3]
i=2: 最小值 3 (下标 4), 交换 → [1, 2, 3, 4, 5]
i=3: 最小值 4 (下标 3), 不交换 → [1, 2, 3, 4, 5]
```

### 3. 冒泡排序

```c
void bubble_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int swapped = 0;
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交换相邻元素
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = 1;
            }
        }
        // 如果没有交换，已经有序
        if (!swapped) break;
    }
}
```

**关键逻辑**：
- 每轮将最大值冒泡到最后
- 相邻元素比较交换
- 如果某轮没有交换，提前结束

**示例**：
```
原数组：[5, 3, 1, 4, 2]

第 1 轮：
比较 5>3, 交换 → [3, 5, 1, 4, 2]
比较 5>1, 交换 → [3, 1, 5, 4, 2]
比较 5>4, 交换 → [3, 1, 4, 5, 2]
比较 5>2, 交换 → [3, 1, 4, 2, 5]

第 2 轮：
比较 3>1, 交换 → [1, 3, 4, 2, 5]
比较 3<4, 不交换
比较 4>2, 交换 → [1, 3, 2, 4, 5]

第 3 轮：
比较 1<3, 不交换
比较 3>2, 交换 → [1, 2, 3, 4, 5]

第 4 轮：没有交换，提前结束
```

## 复杂逻辑解释

### 直接插入排序为什么是稳定的？

```
原数组：[3a, 3b, 1]

插入 3b 时：
- 比较 3a > 3b? 不是（相等）
- 停止，3b 插入到 3a 后面

结果：[1, 3a, 3b]  // 保持了 3a 和 3b 的相对顺序
```

### 简单选择排序为什么不稳定？

```
原数组：[3a, 3b, 1]

第 1 轮：
- 最小值 1 (下标 2)
- 交换 3a 和 1 → [1, 3b, 3a]

结果：[1, 3b, 3a]  // 3a 和 3b 的相对顺序改变了
```

### 冒泡排序为什么是稳定的？

```
原数组：[3a, 3b, 1]

第 1 轮：
- 比较 3a > 3b? 不是（相等），不交换
- 比较 3b > 1? 是，交换 → [3a, 1, 3b]

结果保持了 3a 和 3b 的相对顺序
```

## 易错点

| 易错点 | 说明 |
|--------|------|
| 插入排序下标 | 从 i=1 开始，不是 i=0 |
| 选择排序交换条件 | `min != i` 时才交换 |
| 冒泡排序边界 | `j < n-1-i`，不是 `j < n-1` |
| 排序稳定性 | 插入和冒泡稳定，选择不稳定 |

---

## 复杂度总结

| 算法 | 最好 | 最坏 | 平均 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 直接插入 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 简单选择 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 冒泡排序 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
