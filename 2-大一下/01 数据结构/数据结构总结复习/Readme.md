
# 数据结构总结复习

本目录整理了数据结构课程的复习资料，主要面向期末复习和代码题训练。

资料重点围绕：

- 数据结构核心知识点
- C 语言代码实现
- 常见代码题模板
- 易错点总结
- 复杂度分析

## 目录结构

```text
数据结构总结复习/
├── Todo.enhanced.md
├── 模块/
│   ├── 01_xxx/
│   │   ├── Basic.md
│   │   └── PracticeTemplates/
│   │       └── xxx.c
└── 99_notes/
    ├── Mistakes.md
    ├── MustRemember.md
    └── LastWeekReview.md
```

## 使用方式

建议先阅读 `Todo.enhanced.md`，了解复习重点和任务安排。

然后按模块阅读对应的 `Basic.md`，重点掌握其中的代码实现、易错点和复杂度分析。

每个模块下的 `PracticeTemplates/` 存放 C 语言练习代码，代码中保留了 `TODO`，可用于自行补全和复习。
